### 1. applet-select（下拉选择框）

**日**: ラベル＋ドロップダウンコンボ。websms01288 jlbl + jsel パターン。  
**中**: 带标签的下拉选择框，对应 websms01288 的 jlbl + jsel 模式。

#### 使用例 / 使用示例

```html
<applet-select label="担当" name="charge">
  <option value="">選択</option>
  <option value="1">A</option>
  <option value="2">B</option>
</applet-select>
```

#### 動作 / 行为

- **ドロップダウンモード**（`visible-rows` / `size` 未指定時）: カスタムドロップダウンで表示、最大 `visible-rows` 行（デフォルト 8）表示。  
- **リストボックスモード**（`visible-rows` または `size` 指定時）: ネイティブ `<select>` をそのまま表示。  
- **multiple**: 指定時は常にリストボックスモード。

#### 属性一覧 / 属性列表

| 属性 | 日 | 中 | 例 |
|------|-----|-----|-----|
| label | ラベルテキスト | 标签文字 | `"担当"` |
| label-width | ラベル幅 | 标签宽度 | `80px` |
| label-align | ラベル揃え | 标签对齐 | `right` \| `left` |
| hide-label | 指定時ラベル非表示 | 指定时隐藏标签 | （布尔） |
| select-width | セレクト幅 | 下拉框宽度 | `220px` |
| visible-rows | ドロップダウン表示行数（デフォルト 8） | 下拉列表显示行数（默认 8） | `10` |
| disabled | 無効 | 禁用 | （布尔） |
| required | 必須 | 必填 | （布尔） |
| name | フォーム名 | 表单字段名 | `charge` |
| multiple | 複数選択 | 多选 | （布尔） |
| size | 表示行数（リストボックス時） | 显示行数（列表模式） | `8` |
| width, height | サイズ | 尺寸 | `200px` |
| position, left, top | レイアウト | 布局 | `absolute`, `10px` |
| gap | ラベルと入力の間隔 | 标签与输入间距 | `4px` |
| background | 背景色 | 背景色 | `#fff` |
| font-size, font-family | フォント | 字体 | `11px` |
| border-color, padding | ボーダー・余白 | 边框、内边距 | - |
| disabled-background | 無効時背景（未指定時は透明） | 禁用时背景（未指定时透明） | `#D4D0C8` |
| label-font-size, label-color | ラベルスタイル | 标签样式 | - |
| label-border, label-background | ラベル枠・背景 | 标签边框、背景 | - |

#### API

### 2. applet-combobox（可输入下拉框）

**日**: ラベル＋datalist 付き入力コンボ。入力または候補から選択可能。  
**中**: 带标签的可输入下拉框，支持输入或从候选项中选择。

#### 使用例 / 使用示例

```html
<applet-combobox label="担当" list="myList" placeholder="入力または選択"></applet-combobox>
<datalist id="myList">
  <option value="A">A 説明</option>
  <option value="B">B 説明</option>
</datalist>
```

#### 動作 / 行为

- **datalist**: 子要素の `<datalist>` または `list` 属性で指定した id の datalist を候補として使用。  
- **ドロップダウン**: focus/click で表示。最大 `visible-rows` 行（デフォルト 8）表示、超出部分可滚动。  
  - **focus/click 時**（デフォルト）: 全件表示。選択済みでも一覧を再表示可能。  
  - **input 時**: 入力値でフィルタ。  
  - **filter-on-focus 指定時**: focus/click でも入力値でフィルタ（従来の挙動）。  
- **portal**: 親 overflow を避けるため、ドロップダウンは `position:fixed` で body 直下に portal として表示。

#### 属性一覧 / 属性列表

| 属性 | 日 | 中 | 例 |
|------|-----|-----|-----|
| label | ラベルテキスト | 标签文字 | `"担当"` |
| label-width | ラベル幅 | 标签宽度 | `80px` |
| label-align | ラベル揃え | 标签对齐 | `right` \| `left` |
| hide-label | 指定時ラベル非表示 | 指定时隐藏标签 | （布尔） |
| input-width | 入力幅 | 输入框宽度 | `220px` |
| visible-rows | ドロップダウン表示行数（デフォルト 8） | 下拉列表显示行数（默认 8） | `10` |
| list | datalist の id | datalist 的 id | `myList` |
| placeholder | プレースホルダ | 占位符 | `"入力または選択"` |
| disabled | 無効 | 禁用 | （布尔） |
| value | 初期値 | 初始值 | `"A"` |
| maxlength | 最大文字数 | 最大字符数 | `50` |
| autocomplete | オートコンプリート | 自动完成 | `off` |
| required | 必須 | 必填 | （布尔） |
| name | フォーム名 | 表单字段名 | `charge` |
| type | 入力タイプ | 输入类型 | `text` \| `search` |
| width, height | サイズ | 尺寸 | - |
| position, left, top | レイアウト | 布局 | - |
| input-font-size, input-font-family | 入力フォント | 输入框字体 | - |
| input-background, input-border-color | 入力スタイル | 输入框样式 | - |
| input-padding | 入力余白 | 输入框内边距 | - |
| disabled-background | 無効時背景（未指定時は透明） | 禁用时背景（未指定时透明） | `#D4D0C8` |
| filter-on-focus | 指定時 focus/click も入力値でフィルタ。未指定時は全件表示 | 指定时 focus/click 也按输入值过滤。未指定时显示全部 | （布尔） |
| label-border, label-background | ラベル枠・背景 | 标签边框、背景 | - |

#### API

| プロパティ/メソッド | 型 | 説明 (日) | 说明 (中) |
|---------------------|-----|-----------|-----------|
| value | get/set | 入力値 | 输入值 |
| disabled | get/set | 無効状態 | 禁用状态 |
| inputElement | get | 内部 `<input>` 要素 | 内部 input 元素 |
| datalistElement | get | 内部 datalist 要素 | 内部 datalist 元素 |
| refreshOptions() | method | 候補リスト再取得 | 刷新候选项列表 |

#### イベント / 事件

| イベント | 説明 (日) | 说明 (中) |
|----------|-----------|-----------|
| change | 候補選択時、detail.value に選択値 | 选择候选项时，detail.value 为选中值 |

## 六、補足 / 补充说明

### 6.1 Select / ComboBox の visible-rows

- **デフォルト**: 8 行。  
- **実装**: ドロップダウン内の各項目は `min-height: 20px`、`max-height` は `visible-rows × 20px` で制限。  
- **ComboBox**: ドロップダウンは portal（body 直下）に表示されるため、`--combobox-dropdown-max-rows` を portal に渡して表示行数を制御。

### 6.2 FormField.js の役割

- `applet-form-field` と `applet-head-label` の両方を定義。  
- 1 ファイル読み込みで 2 コンポーネントが利用可能。
